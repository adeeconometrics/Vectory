class vector.vector(vector: list)[source]¶

Bases: object

Representation of real-valued vector \(\vec{v} \in \mathbb{R}^n\). The vector type is an iterable object.

Supported Operations:

  • element-wise vector addition

  • element-wise vector subtraction

  • element-wise vector multiplication

  • element-wise vector division

  • scalar-vector multiplication

  • scalar-vector division

  • vector-scalar exponentiation

>>> import vector as v
>>> a = v.vector([1,2,3])
>>> b = v.vector([4,5,6])
>>> c = 5
>>> a+b
[5, 7, 9]
>>> a-b
[-3, -3, -3]
>>> a*b
[4, 10, 18]
>>> a/b
[0.25, 0.4, 0.5]
>>> a*c
[5, 10, 15]
>>> a/c
[0.2, 0.4, 0.6]
>>> a**c
[1, 32, 243]
